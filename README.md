## 📌 Tentang Dead Desk
Dead Desk adalah game simulasi naratif/manajemen berbasis teks yang unik. Di sini, pemain mengambil peran sebagai entitas supranatural (Dewa Kematian) yang duduk di balik meja birokrasi akhirat. Tugas Anda? Memeriksa dokumen rekam jejak kehidupan manusia dan memutuskan nasib akhir arwah mereka.

### ✨ Fitur Utama
* **Document Inspection:** Membaca biodata, dosa, dan kebaikan dari arwah yang datang ke meja kerja Anda.
* **Moral Dilemmas:** Menghadapi kasus-kasus ambigu yang menguji objektivitas dan empati Anda sebagai hakim akhirat.
* **Branching Consequences:** Setiap keputusan sortir yang Anda buat akan memengaruhi keseimbangan dunia akhirat dan membuka *ending* game yang berbeda.
* **Immersive Dark-Theme UI:** Desain antarmuka monokromatik/gelap yang mendukung atmosfer misterius dan dingin khas perkantoran akhirat.

### 🛠️ Tech Stack
* **Core:** JavaScript (Game logic, JSON-based storylines), HTML5, CSS Variables (Dark-theme layout).
